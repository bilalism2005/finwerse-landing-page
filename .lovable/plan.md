
# Finverse Landing Page

Build a single-page landing site for Finverse following the Matiks-inspired dark theme with neon green accents, bold condensed typography, sticker-style icons, and alternating Z-pattern layout.

## Sections to Build

### A. Fixed Header
- Left: Finverse logo
- Center-right: "Home" and "Solutions" links (white text)
- Far right: "Try Now" CTA button (transparent, thin white border, rounded)

### B. Hero Section
- Dark background with subtle grey geometric grid pattern
- Left: 3D smartphone mockup placeholder showing Finverse app UI
- Right: Large bold text "TURN RANDOM TIPS INTO VERIFIED TRADES" (white), subheading "AI-powered scores for every stock." (grey), black "TRY NOW" button with neon-green drop shadow

### C. Pain & Reassurance Sections (7 blocks, alternating Z-pattern)
Each block has a sticker-style icon and text, alternating left/right placement:
1. "TRUSTED RANDOM TRADING CALL, BOOKED REAL LOSS." → "You don't need to stop taking calls. Just start verifying them."
2. "MARKET DIPPED. EVERYTHING FELL. NOT UNLUCKY. YOU WERE UNDIVERSIFIED." → "Building a resilient portfolio isn't hard. It just needs the right information."
3. "LATE ENTRY. MISSED THE MOVE. / WAITED FOR MORE. LEFT WITH LESS." → "Mistimed entries and exits are universal. Alerts make them avoidable."
4. "RESEARCH TOOK 2 HOURS. CONFIDENCE WAS STILL ZERO." → "One prompt. No more hours. Just verified answers."
5. "TRIED TO RECOVER. WENT DEEPER IN RED." → "The urge to recover is human. Now there's a way to control it."
6. "CHART PATTERNS FELT LIKE EXPERT STUFF." → "Not anymore."
7. "NEWS FELT BIG. STOCK DIDN'T MOVE." → "Headlines are designed to feel urgent. The score tells you which ones to act on."

### D. Solution Sections (7 blocks, alternating Z-pattern with image placeholders)
1. Stock Analytics: "VERIFIED TRADES ONLY. EVERYTHING ELSE STAYS OUT." + 3 bullets
2. Portfolio Health: "PORTFOLIO THAT HOLDS WHEN OTHERS FALL." + 3 bullets
3. Entry-Exit Alerts: "ALWAYS ON TIME. EVERY TRADE." + 3 bullets
4. AI Chatbot: "INSTANT CLARITY. HOURS SAVED." + 3 bullets
5. Impulse Analyzer: "FEWER EMOTIONAL TRADES. PORTFOLIO RECOVERS." + 3 bullets
6. Chart Analyzer: "BE YOUR OWN ANALYST." + 3 bullets
7. Focus Feed: "UNBOTHERED BY HEADLINES." + 3 bullets

### E. Pre-Footer
- 2×2 grid of social icons (Discord, X, Instagram, YouTube) as white rounded cards
- Green card with bold text "LET'S TRADE ON DATA. NOT ON TIPS." + "TRY NOW" button

### F. Footer
- Light purple rounded-top background
- Left: "FINVERSE" in large bold black text
- Right: Stacked links — "BACK TO TOP ↗", "TERMS AND CONDITION ↗", "PRIVACY POLICY ↗"

## Design System
- **Colors**: Black (#1a1a1a) background, neon green (#b8ff00) accents, white headings, muted grey (#9ca3af) subtext, light purple (#c4b5fd) footer
- **Typography**: Heavy condensed sans-serif (Inter/Impact style) for headings, all-caps; lighter weight for body text
- **Icons**: Flat vector sticker-style illustrations with thick white borders (using Lucide icons styled with colored backgrounds and white strokes as placeholders)
- **Grid pattern**: CSS-based subtle geometric grid overlay on hero only
- **Layout**: Fully responsive, Z-pattern alternating sections
- **Animations**: Smooth scroll, subtle fade-in on scroll for sections
